## Key features

<Columns cols={3}>
  <Card title="PIX payments" icon="bolt">
    Instant payments via Brazil's PIX network with QR codes and copy-paste.
  </Card>
  <Card title="Multiple methods" icon="credit-card">
    Accept PIX, credit cards, boletos, and Litecoin.
  </Card>
  <Card title="Split payments" icon="arrows-split-up-and-left">
    Split payments across up to 10 recipients in a single transaction.
  </Card>
  <Card title="Hosted checkout" icon="browser">
    Use YuvexPay's hosted checkout page or build your own with headless mode.
  </Card>
  <Card title="Webhooks" icon="webhook">
    Real-time event notifications with automatic retries and HMAC signatures.
  </Card>
  <Card title="Sandbox" icon="flask">
    Test your integration with sandbox credentials before going live.
  </Card>
